How to Take the GenAI Training

This page will guide you through the steps to successfully complete the GenAI Engineer 6-week training program.

Step-by-Step Guide

  1. Review the weekly plan on the main page to understand the structure and outcomes for each week.
  2. Complete all assigned courses and watch the recommended videos for each week.
  3. Work through the hands-on labs to gain practical experience with the tools and frameworks.
  4. Finish the mini-project for each week and push your code to a GitHub repository you create in your own account.
  5. Take the evaluation quiz at the end of each week to assess your understanding. You must score at least 80% to pass.
  6. Repeat the above steps for all 6 weeks. At the end, complete the capstone project as described in the main plan.

Best Practices

  • Keep your GitHub repository updated with your weekly mini-projects and capstone work.
  • Document your code and learning journey in your repo's README.
  • Reach out for support if you get stuck or need clarification.

Prerequisites (for all participants)

Modern laptop (16GB+ RAM), Python basics, and ability to install software required.
  • Laptop & System: Modern laptop (Mac/Windows/Linux) with 16GB RAM minimum (for local LLMs), 50GB+ free disk, reliable internet, admin rights, webcam & mic.
  • Software: Python 3.11+, VS Code, Git & GitHub, browser, ability to install Docker (optional), Jupyter (optional).
  • Programming: Basic Python (variables, functions, classes, file I/O), command line/shell basics, virtual environments (venv/Poetry), basic git usage.
  • Other: Willingness to learn new frameworks (LangChain, CrewAI), basic AI/ML concepts (helpful), comfort with docs & troubleshooting.
  • Optional: Experience with Jupyter, REST APIs, web dev, SQL/databases.

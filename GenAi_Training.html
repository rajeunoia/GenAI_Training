GenAI Engineer — 6-Week Training Plan

Team size: 6–8 freshers • Stack: Ollama → Bedrock → Azure OpenAI • Vector DB: Chroma • Frameworks: LangChain (primary), CrewAI (secondary) • Protocols: MCP, ACP/A2A • Includes capstone

Week 1 — Foundations & Local Setup

Outcome: Run local LLMs, practice solid prompting, and ship a LangChain “hello chain”.

Watch / Study

  • AI Python for Beginners (DeepLearning.AI)
  • ChatGPT Prompt Engineering for Developers (DeepLearning.AI)
  • Generative AI with LLMs (Coursera) — foundation modules

Additional Videos

  • Learn Ollama in 15 Minutes – Run LLM Models Locally for FREE (YouTube)
  • LangChain Explained in 13 Minutes — QuickStart Tutorial (YouTube)
  • Get Started with GitHub Copilot in VS Code (≈30 min) (YouTube)

Hands-on Labs

  • Install Ollama; pull a Llama 3.x model; run local prompts from Python.
  • Set up project scaffolding: Python 3.11+, venv/Poetry, Ruff, Pytest.
  • LangChain intro: PromptTemplate → LLM → structured JSON outputs.

Mini-Project #1

  • Local Q&A over a single file using Ollama; configurable prompt library.

Week 2 — Vector Databases & RAG

Outcome: Build a working RAG pipeline with Chroma; optionally swap local ↔ cloud models.

Watch / Study

  • Vector Databases: From Embeddings to Applications (DeepLearning.AI)
  • LangChain for LLM Application Development (DeepLearning.AI)

Additional Videos

  • Complete ChromaDB Tutorial with LangChain (YouTube)

Hands-on Labs

  • Stand up ChromaDB; embed documents; similarity search & basic reranking.
  • Wire a simple RAG chain (Retriever → LLM → answer post-processing).

Mini-Project #2

  • KB assistant over team docs using Chroma + Ollama; switch to Bedrock/Azure OpenAI for comparison.

Week 3 — Tool Use & Database Integration

Outcome: Agents call tools and answer questions by executing SQL safely.

Watch / Study

  • LangChain for LLM Application Development (finish agents & tools)
  • Generative AI with LLMs (Coursera) — modeling lifecycle & evals modules
  • Building Your Own Database Agent (DeepLearning.AI)

Hands-on Labs

  • Create a SQL tool (SQLite/Postgres) with LangChain’s SQLDatabase chain.
  • Combine DB tool + RAG tool in a single agent; JSON-schema outputs & error handling.

Mini-Project #3

  • “Data Q&A Agent” that routes to DB or KB depending on the question.

Week 4 — Agents, Multi-Agent Patterns & Evaluation

Outcome: Multi-tool agent with evaluation and basic observability.

Watch / Study

  • Multi-AI Agent Systems with CrewAI (DeepLearning.AI)
  • Evaluating AI Agents (DeepLearning.AI)

Additional Videos

  • Build a Smart AI Agent with LangGraph & Python (≈10 min) (YouTube)

Hands-on Labs

  • Plan-&-Execute agent in LangChain with 2–3 tools (RAG, SQL, web).
  • Add a small golden set for evals; tracing comes in Week 6.

Mini-Project #4

  • “Research Agent” that plans tasks, calls tools, and returns a structured brief with references.

Week 5 — Protocols: MCP & ACP (A2A)

Outcome: A local MCP server + an ACP exchange between two agents.

Watch / Study

  • MCP: Build Rich-Context AI Apps with Anthropic (DeepLearning.AI)
  • ACP: Agent Communication Protocol (DeepLearning.AI)
  • A2A Agent-to-Agent Protocol (Udemy)

Additional Videos

  • Build an MCP Server in 20 Minutes (with Python) (YouTube)

Hands-on Labs

  • Build a local MCP server exposing a DB or KB tool; test via MCP-capable client.
  • Implement an ACP client↔server message exchange; agent handoff (research → summary).

Mini-Project #5

  • Agent interacts with MCP tool and hands off a task to another agent via ACP/A2A.

Week 6 — Integration, Finetuning Basics & Capstone

Outcome: End-to-end build: RAG + SQL + MCP + ACP/A2A with evals & minimal guardrails.

Watch / Study

  • Finetuning Large Language Models (DeepLearning.AI)
  • Generative AI with LLMs (Coursera) — reinforcement & app patterns (optional focus)

Additional Videos

  • Add Observability to Your AI-Powered Applications (Langfuse) (YouTube)
  • NeMo Guardrails – Tame your LLM without Prompt Engineering (YouTube)
  • Learn AWS Bedrock in 20 Minutes (YouTube)
  • Getting Started with Azure OpenAI (2025 quickstart) (YouTube)

Capstone (team of 2–3)

  • Repo with: RAG (Chroma) + SQL tool (LangChain) + MCP tool + ACP/A2A handoff.
  • Include tracing hooks, basic guardrails (PII redaction / content filter), README and demo script.
